Este proyecto tiene como objetivo aprender a utilizar Git y GitHub para desarrollar proyectos de forma colaborativa y llevar un control de versiones del código.

Descripción de la carrera

La carrera de Tecnologías de la Información forma profesionales capaces de diseñar, desarrollar, administrar y mantener sistemas informáticos. Los estudiantes adquieren conocimientos en programación, redes, bases de datos, seguridad informática y desarrollo de software para crear soluciones tecnológicas que respondan a las necesidades de empresas y organizaciones.
