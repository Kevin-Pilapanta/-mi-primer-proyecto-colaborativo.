# -mi-primer-proyecto-colaborativo.

## Objetivo del proyecto
Aprender a utilizar Git y GitHub para desarrollar proyectos colaborativos.

## Integrante

## Herramientas colaborativas investigadas
- Git
- GitHub
- visual studio code
## Mejoras
GitHub es la plataforma en la nube más importante para el desarrollo de software. Funciona como un "Google Drive para código" donde se aloja el código fuente de miles de programas, permitiendo que programadores de todo el mundo colaboren sin sobreescribir el trabajo de los demás y manteniendo un historial de cada cambio.
